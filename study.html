Ace Academy

Application featuring automated term-definition mapping with multiple study tools.

Description:

Ace Academy is a web-based tool designed to streamline the way students study terms and definitions. Developed as a group project, this application focuses on providing an interactive and shareable environment for custom study material.

    Key Capabilities:
  • Custom Study Sets: A simple creation tool where you can name a set and quickly add as many term-and-definition pairs as you need.
  • Flashcards & Dynamic Quizzes: Once a set is ready, you can choose between traditional front-and-back flashcards for memorization or a Dynamic Quiz mode that tests your knowledge more interactively.
  • JSON Export & Import: To make studying collaborative, we built a feature that lets you export your sets as JSON files. You can share these files with friends, who can then import them directly into the site to start studying your material instantly.
Tech used: HTML | CSS | JavaScript
