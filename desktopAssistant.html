AI Desktop Assistant

Voice-controlled assistant using Mistral AI and Selenium to automate desktop and browser tasks

Description

This project is an intelligent "action-oriented" agent designed to bridge the gap between conversation and execution. By combining Natural Language Processing (NLP) with system-level automation, the assistant functions as both a conversational AI and a hands-free desktop controller.

    Key Capabilities:
  • Conversational Intelligence: Beyond executing tasks, the assistant acts as a standard LLM interface, answering complex questions and providing information just like a traditional AI.
  • Hybrid Interface: Built a modern CustomTkinter UI featuring a dedicated chat window for text-based interaction and a real-time system log to monitor background processes and command execution.
  • Web Automation: Using the Selenium library, the program can perform live web searches, navigate pages, and manage browser tabs entirely through verbal commands.
  • System Execution: It translates natural language into executable commands, allowing for seamless control over computer functions without the need for a keyboard or mouse.
Tech used: Python | Mistral AI | Selenium | CustomTkinter | PyAutoGUI
