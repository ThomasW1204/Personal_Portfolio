Drone Security System

Autonomous drone system utilizing facial recognition for perimeter patrol and security monitoring

Description:

Developed as a collaborative group project, this system transforms a Tello drone into an automated security agent. By bridging the gap between computer vision and robotics, our team created a hands-free surveillance solution capable of autonomous patrolling and biometric verification.

    Key Capabilities:
  • Autonomous Patrol: Programmed the drone to execute a precision square patrol pattern, maintaining consistent spatial awareness while providing real-time coordinate tracking to a web-based frontend.
  • Biometric Verification: Integrated OpenCV and facial recognition logic to handle "Check-In" requests. Upon activation, the drone autonomously navigates to the user, aligns its camera for a biometric scan, and validates the user against a database.
  • Integrated Command Center: Developed a comprehensive dashboard featuring a live camera stream, a real-time spatial simulation, and a system console for mission logging and status updates.
  • State Machine Logic: Engineered a robust mission-loop that allows the drone to seamlessly transition from patrolling to check-in mode and back, ensuring uninterrupted surveillance coverage.

Tech used: Java | OpenCV | Java Swing
